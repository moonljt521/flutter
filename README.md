# 一个flutter入门的demo，集成了一些功能，包括：
 listview 的使用；  
 请求的使用；  
 bottomNavigator的使用；  
 router 的使用和封装；  
 drawer 的使用；  
 简单的页面切换转场动画的使用；  
 flutter 调用android原生的功能，目前只写了一个调用原生的toast，其他的类似；  
 正式打apk包的功能  
 其中drawer里包含了两个小demo，一个是计算器一个是单词收藏列表
